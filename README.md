# Aplikasi SPP

Petunjuk Instalasi
==================
1.  Rename file env menjadi .env (titik env)
2.  Ubah bagian 

    app.baseURL = ''

    Menjadi alamat URL aplikasi anda
3.  Ubah environment menjadi production jika akan di onlinkan
4.  Ubah seting database di bagian Database
